GUI TURBO ASSEMBLER VER. 1.0

Thank you for installing this application. GUI Turbo Assembler Ver. 1.0 is a small graphical utility to improve
user interface and interaction while using Turbo Assembler.

This package comes with Turbo Assembler 4.1 and Turbo Linker 7.1.

This is developed using Microsoft .NET Framework 4.0, so you should install it before using this.
If you don't have the framework, you can download it from the link given below

http://www.microsoft.com/en-in/download/details.aspx?id=17718
